# Student Registration System

## Project Description
This project is a Student Registration System built using HTML, CSS, and JavaScript.  
It allows users to add, edit, and delete student records.  
The project also stores data in localStorage, so the student details remain available even after refreshing the page.

## Features
- Add new student records
- Edit existing student records
- Delete student records
- Form validation for:
  - Student name (only characters)
  - Student USN / ID (only numbers)
  - Valid email format
  - Contact number (minimum 10 digits)
- Data saved in browser localStorage
- Responsive design for:
  - Mobile
  - Tablet
  - Desktop
- Dynamic vertical scrollbar for student records table

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## How to Use
1. Enter student details in the form.
2. Click **Add Student** to save a new record.
3. Click **Edit** to modify an existing record.
4. Click **Delete** to remove a record.
5. Refresh the page to see that data is still stored using localStorage.

## Validation Rules
- Student name should contain only letters
- Student USN / ID should contain only numbers
- Email should be in valid format
- Contact number should contain at least 10 digits
- Empty fields are not allowed
